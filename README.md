后台以 PHP编程进行搭建
====================================================
           运营控制台 - 功能与使用说明书
====================================================

一、 📊 数据大屏浏览（包含流量监控）
这里是你的“指挥中心”。系统会自动统计当前上架了多少个产品、建立了多少个代理通道、配置了多少条映射规则等。
* 重点防线：这里会清晰显示你“当前的公网 IP”，当你需要把自己的 IP 加入白名单时，直接看这里即可。
* 流量走势：提供近 7 天的全站总流量走势（PV/UV）以及今日 24 小时的流量分布图表。鼠标悬停可查看精确数据。为保障系统极速运行，30 天前的历史统计数据会自动无痕清理。

二、 🏆 流量风云榜
精准查询各大代理商的推广效果与业绩。
* 自由查询：支持日历和时间段选择。你可以随意查看最近 30 天内【任意一天】甚至【精确到某一个小时】的各通道访问点击量 (PV) 和独立访客数 (UV) 排行榜。

三、 👣 访客访问日志
实时追踪前端客户的访问足迹。
* 精确记录：记录访客的精确访问时间、真实 IP 地址以及受访的通道代号。
* 快捷筛选：支持通过下拉框，单独过滤出某个指定代理通道的访客记录。
* 自动瘦身（防卡顿）：系统仅保留最近 7 天的记录，并在每次加载时自动在底层物理删除过期数据，页面最多展示最新 500 条，永远不会拖垮服务器硬盘。

四、 🗂️ 分类板块管理
用来管理前端主页上的大分类（例如：8NN集团产品、电子模拟器等）。
* 添加：输入名字直接添加。即使分类下暂时没有产品，前端也会正常显示该分类标签。
* 修改：直接在列表里改名字，点“保存修改”。
* 注意：如果彻底删除了某个分类，该分类下面的所有产品也会被一起清空！

五、 🎁 产品管理
在这里把所有的游戏或产品上架到对应的“分类板块”里。
* 平台默认链接：指当“普通散客”（没有使用任何代理后缀的访客）打开网站时，点击该产品跳去的地址。
* 图标：可以点击“选择文件”上传电脑里的图，也可以直接填网络图片的 URL。系统配备智能寻路引擎，自动防破图。

六、 🔀 通道管理（代理商管理）
在这里给你的代理商开通专属推广通道。
* 通道代号：必须是英文或数字（例如：4g56sg、agent88）。这就是代理商以后用来推广的“暗号”。
* 代理商名称：你自己备注用的名字。支持随时修改名称，系统会自动同步以前的映射数据。
* 推广方式：代理商只需让客户访问 【你的主域名/?code=通道代号】 即可。

七、 🔗 通道映射管理（核心功能！）
产品上架了，通道也建好了，必须在这里“牵线搭桥”。
* 功能：为指定产品分配【代理通道】，并设置该代理的【专属跳转域名】。
* 核心拦截逻辑：如果客户用 ?code=4g56sg 访问，前端网页只会显示在这里配置了 4g56sg 的专属产品。没配置 4g56sg 的产品，对该客户全部强制隐身！
* 跳转逻辑：客户点击产品后，会跳到你在这里配置的“专属跳转域名”，而不是平台默认链接。
* 快捷操作：点击“📦 批量设置映射”，可以一次性勾选多个产品，给它们批量填入某个代理的专属域名。弹窗内附带快捷搜索框，防眼花找错通道。

八、 🖼️ 轮播图管理
管理主页顶部的滚动大图（建议尺寸 790x318）。
* 支持双轨添加：你可以直接上传本地图片，也可以粘贴外链图片地址。上传后下方会展示缩略图，点击“彻底删除”即可下架。

九、 🔗 底部快捷链接编辑
修改主页底部的三个长条横幅（如：爆庄必备心法、优惠大厅等）。
* 文本：自由修改标题和副标题。
* 图标：支持本地上传，也支持网络外链。
* 跳转：如果你只想让它展示，不想让客户点击跳转，把“被点击时跳转的域名”【留空】或者填【#】即可。

十、 📝 全局配置
主页上的所有零碎文本都在这里改。
* 网站标题、滚动跑马灯公告、主LOGO（支持直接上传覆盖并预览现有图片）、防骗提示的标题和内容、底部的版权年份等。改完点保存，前端刷新立刻生效。
* 备注：网页最顶部的标签名字（<title>）建议直接在 index.html 文件里手动修改，更有利于搜索引擎收录。

十一、 🛡️ IP白名单管理（后台安全门禁）
最高级别的安全防护墙！
* 原理：只要这个列表里有 IP，系统就会自动锁死。所有不在列表里的人，访问后台全都会显示“403 Forbidden（拒绝访问）”。
* 建议：请务必确保你当前的 IP（在“数据大屏”里能看到）已经加在里面了。如果公司有多台电脑，要把它们的 IP 都加上。

十二、 🆘 紧急防封锁 / 自助加白系统 (whitelist.php)
如果你因为手机网络切换、路由器重启、开启了加速器等原因导致你的公网 IP 变了，被系统无情拦截（显示 403 Forbidden）进不去后台了，怎么办？
别慌！系统配有“自助加白”秘密通道：
1. 在浏览器手动打开安全链接：【你的主域名/whitelist.php】（或者在 admin 目录下则访问 /admin/whitelist.php）
2. 此时会看到一个被锁住的页面，在页面上方的密码框中，输入你的【IP加白专属安全密码】（默认是 999999，只有最高管理员知道，可在 config.php 中修改）。
3. 点击【👁️ 解锁列表】，系统验证密码正确后，会自动显示你当前的最新真实 IP。
4. 复制你的最新 IP 填入输入框，点击【➕ 强制放行此 IP】。
5. 提示加白成功后，重新打开 admin.php，你就可以顺利进入后台继续操作了！
