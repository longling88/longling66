后台以 PHP进行搭建
====================================================
           运营控制台 - 功能与使用说明书
====================================================

一、 📊 数据大屏概览
这里是你的“指挥中心”。系统会自动统计当前上架了多少个产品、建立了多少个代理通道、配置了多少条映射规则等。
* 重点：这里会显示你“当前的登录 IP”，当你需要把自己的 IP 加入白名单时，直接看这里即可。

二、 🗂️ 分类板块管理
用来管理前端主页上的大分类（例如：8NN集团产品、电子模拟器等）。
* 添加：输入名字直接添加。
* 修改：直接在列表里改名字，点“保存修改”。
* 注意：如果彻底删除了某个分类，该分类下面的所有产品也会被一起清空！

三、 🎁 产品管理
在这里把所有的游戏或产品上架到对应的“分类板块”里。
* 平台默认链接：指当“普通散客”（没有使用任何代理后缀的访客）打开网站时，点击该产品跳去的地址。
* 图标：可以点击“选择文件”上传电脑里的图，也可以直接填网络图片的 URL。两者选其一即可。

四、 🔀 通道管理（代理商管理）
在这里给你的代理商开通专属通道。
* 通道代号：必须是英文或数字（例如：4g56sg、agent88）。这就是代理商以后用来推广的“暗号”。
* 代理商名称：你自己备注用的名字，方便你在后台认人。
* 推广方式：代理商只需让客户访问 【你的主域名/?code=通道代号】 即可。

五、 🔗 通道映射管理（核心功能！）
产品上架了，通道也建好了，必须在这里“牵线搭桥”。
* 功能：为指定产品分配【代理通道】，并设置该代理的【专属跳转域名】。
* 逻辑：
  1. 如果客户用 ?code=4g56sg 访问，系统只会显示在这里配置了 4g56sg 的产品。
  2. 没配置 4g56sg 的产品，对该客户全部隐身！
  3. 客户点击产品后，会跳到你在这里配置的“专属跳转域名”，而不是平台默认链接。
* 快捷操作：点击“📦 批量设置映射”，可以一次性勾选多个产品，给它们批量填入某个代理的专属域名。右侧带有双重搜索框，方便快速找人找域名。

六、 🖼️ 轮播图管理
管理主页顶部的滚动大图（建议尺寸 790x318）。
* 支持双轨添加：你可以直接上传本地图片，也可以粘贴外链图片地址。上传后下方会显示缩略图，点击“彻底删除”即可下架。

七、 🔗 底部快捷链接编辑
修改主页底部的三个长条横幅（如：爆庄必备心法、优惠大厅等）。
* 文本：自由修改标题和副标题。
* 图标：支持本地上传，也支持外链。
* 跳转：如果你只想让它展示，不想让客户点击跳转，把“被点击时跳转的域名”【留空】或者填【#】即可。

八、 📝 全局配置
主页上的所有零碎文字都在这里改。
* 网站标题、滚动跑马灯公告、主LOGO（直接上传覆盖）、防骗提示的标题和内容、底部的版权年份等。改完点保存，前端刷新立刻生效。

九、 🛡️ IP白名单管理（安全门禁）
最高级别的安全防护墙！
* 原理：只要这个列表里有 IP，系统就会自动锁死。所有不在列表里的 IP，访问后台全都会显示“403 Forbidden（拒绝访问）”。
* 建议：请务必确保你当前的 IP（在“数据概览”里能看到）已经加在里面了。如果公司有多台电脑，要把它们的 IP 都加上。
* 强拆方法：如果你不小心把自己锁在外面了，需要登录服务器终端，用代码编辑器（如 nano）打开 admin.php 或 whitelist.json，手动把你的新 IP 加进去即可。
