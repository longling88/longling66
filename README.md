运营控制台 - 核心功能与使用说明书
欢迎使用运营控制台（后台管理系统）。本系统旨在帮助您高效管理网站的所有内容与配置，涵盖首页展示、产品上架、渠道推广、数据监控等全流程。请仔细阅读以下说明，快速上手各项功能。

1. 登录与退出
登录
访问 admin.php 自动跳转到登录页。

输入管理员账号、密码以及算术验证码（防机器人）。

点击“安全登录系统”即可进入控制台。

退出
点击右上角“安全退出”按钮，清除登录状态并返回登录页。

2. 数据大屏浏览（仪表板）
模块标识: dashboard

本页面实时呈现网站整体运营概况，是您掌握全局的第一站。

统计卡片：显示分类数量、产品总数、通道数、映射数、轮播图数、底部链接数、白名单IP数以及您当前的公网IP。

今日/昨日/前日流量：展示最近三天的访问量（PV）和独立访客数（UV）。

近7天走势图：折线图展示过去7天的 PV/UV 变化趋势。

今日24小时分布图：柱状图展示今日每个小时的 PV/UV 分布，方便发现高峰时段。

所有图表数据来自 stats.json，系统自动记录每次前端访问。

3. 流量风云榜
模块标识: traffic_ranking

查看各推广通道的流量排名，分析渠道效果。

筛选条件：

选择日期（最近30天内）

选择时间段（全天汇总或具体某个小时）

排名表格：展示每个通道的 PV（点击量）和 UV（独立访客数），按 PV 降序排列。

用途：评估各代理或推广渠道的贡献，指导资源配置。

4. 访客访问日志
模块标识: access_logs

记录并检索所有访客的访问明细，便于追溯和审计。

自动清理：系统默认只保留最近7天的日志，过期的自动删除。

手动清理：可选择保留最近1/2/3/7天，点击“确认清理”立即生效。

查询过滤：

按通道代号筛选

按IP地址搜索（支持部分匹配）

展示：每条记录包含访问时间、访客IP、受访通道代号，最多显示最新的500条。

5. 分类板块管理
模块标识: tabs

管理首页的产品分类选项卡（例如“8NN集团产品”、“电子模拟器”等）。

添加分类：输入分类名称，点击“立即创建”。

编辑分类：在列表中直接修改名称，点击“保存”。

设置默认：非默认分类旁边有“设为默认”按钮，点击后该分类成为默认首个选项卡。

排序：点击“上移/下移”调整分类在前端的显示顺序。

删除：只有非默认且非唯一分类可删除，删除后该分类下的所有产品也会被移除（请谨慎操作）。

6. 产品管理
模块标识: product

上架、编辑、删除和调整产品，每个产品隶属于一个分类板块。

添加产品：

选择“所属板块”

填写“产品名称”

填写“平台默认跳转链接”（普通散客点击时跳转的URL）

上传图标或填写图标外链（可选）

点击“上架”

系统自动生成内部标识（用于后端识别），无需手动填写。

产品列表：

按板块筛选：点击上方板块按钮，只显示该板块的产品。

修改产品：点击“修改”按钮，在弹出的窗口中编辑名称、链接、图标。

排序：点击“上移/下移”调整产品在板块内的显示顺序。

删除：彻底移除产品，同时清除该产品的所有通道映射关系。

7. 通道管理
模块标识: channels

管理代理商/推广渠道（即带有 ?code=xxx 的访问链接）。

添加通道：输入通道代号（仅英文+数字）和代理商备注名称，点击“创建通道”。

编辑通道：在列表中直接修改代理商名称，点击“修改名称”保存。

删除通道：删除后，所有使用该通道的映射配置将失效，且该通道的统计数据会被清理。

搜索通道：在搜索框中输入通道代号或代理商名称，表格自动过滤显示匹配的通道，方便快速查找。

8. 通道映射管理
模块标识: channel_mapping

为具体产品绑定代理商通道，并设置该通道下的专属跳转域名。未配置映射的产品在该通道下将不显示（实现按渠道定制产品）。

新增映射：点击“➕ 新增产品映射”，选择产品、选择通道、填入该通道的专属跳转域名，保存即可。

批量设置映射：点击“📦 批量设置映射”，选择一个通道，然后勾选多个产品，并分别为每个产品填写跳转域名，一次性应用。

查看映射列表：表格展示每个产品已配置的通道及对应的专属域名。

删除映射：点击映射行内的“删除”即可移除该产品在该通道下的配置。

搜索过滤：支持按通道代号或跳转域名搜索，快速定位。

去重机制：同一产品同一通道只能有一个映射，新配置会覆盖旧配置，避免重复。

9. 轮播图管理
模块标识: carousel

管理首页顶部的轮播广告图。

添加轮播图：支持本地上传图片或输入图片外链URL，点击“上传/添加”即可。

更换图片：点击轮播图下方的“更换”按钮，可上传新图或填写新链接。

排序：点击“前移/后移”调整轮播图的显示顺序。

删除：移除不需要的轮播图。

建议尺寸：宽790px，高318px。

10. 底部链接编辑
模块标识: banners

管理首页底部的快捷横幅（可配置图标、标题、副标题及跳转链接）。

添加横幅：填写标题、副标题、跳转URL（留空或填#则不跳转），并可上传图标或填写图标外链。

编辑横幅：在列表中直接修改各项内容，点击“保存此横幅修改”。

排序：使用“上移/下移”调整显示顺序。

删除：移除横幅。

11. 全局配置
模块标识: site

设置网站的整体样式和内容。

网站标题：浏览器标签页标题。

左上角LOGO：支持本地上传或填写图片URL。

左上角文字：主标文字一（浅色）和主标文字二（绿色加粗）。

滚动公告：首页顶部滚动的文字。

防骗提示：底部防骗区块的标题和详细内容。

版权信息：底部版权声明。

所有修改点击“保存配置”立即生效。

12. IP白名单管理
模块标识: whitelist

控制哪些IP可以访问后台，提升系统安全性。

显示当前IP：页面顶部会显示您当前访问的公网IP，并提供“添加到白名单”按钮，一键加白。

手动添加：在输入框中填写IP地址，点击“加白”。

移除IP：点击列表中的“移除拦截”，该IP将立即失去后台访问权限（请勿移除自己的IP，否则会被立即踢出）。

注意：白名单数据存储在 admin/whitelist.json 中，与 whitelist.php 共用同一文件。若您使用 whitelist.php 辅助加白，操作会自动同步。

13. 常见问题与注意事项
Q1: 为什么我加白了IP，刷新后仍显示403？
检查是否保留了文件中第二个IP检查点（已在新版中移除）。如果您仍遇到，请联系技术支持，确保您的 admin.php 使用最新版本（只保留第一个检查点）。

Q2: 我添加的产品在后台显示正常，但前端看不到？
确认该产品是否配置了当前通道的映射（若访问链接带 ?code=xxx，则必须为该通道配置映射才会显示）。

检查产品是否被分配到了正确的分类板块，且该板块在 tabs 中正常启用。

Q3: 通道映射重复或显示错误怎么办？
系统已内置自动去重机制，每次保存时会覆盖旧的映射，不会产生重复。如果遇到异常，可手动删除后重新添加。

Q4: 数据大屏的流量数据不准或为空？
确保前端 index.html 正确调用 admin/get_public_data.php，且 stats.json 文件可写。正常访问首页会触发统计记录。

Q5: 如何恢复误删的数据？
系统没有回收站功能，请谨慎操作。建议定期备份 data.json 和 whitelist.json。
